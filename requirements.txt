# Requirements

List of all Python dependencies for the project.

## Core Dependencies

torch==2.0.0
torch-geometric==2.3.0
scikit-learn==1.3.0
pandas==2.0.0
numpy==1.24.0
matplotlib==3.7.0
seaborn==0.12.0

## Installation

```bash
pip install -r requirements.txt
```

## What Each Package Does

- **torch** (2.0.0): Deep learning framework for neural networks
- **torch-geometric** (2.3.0): Graph neural network library
- **scikit-learn** (1.3.0): Machine learning utilities (metrics, preprocessing)
- **pandas** (2.0.0): Data manipulation and analysis
- **numpy** (1.24.0): Numerical computing
- **matplotlib** (3.7.0): Plotting and visualization
- **seaborn** (0.12.0): Statistical data visualization

## Optional Packages

For GPU acceleration (NVIDIA CUDA):
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

For Jupyter notebook support:
```bash
pip install jupyter notebook
```

For development:
```bash
pip install black flake8 pytest
```
